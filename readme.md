# Usage
<!-- usage -->
```sh-session
$ npm install -g shower-cli
$ shower COMMAND
running command...
$ shower (-v|--version|version)
shower-cli/0.2.2 darwin-x64 node-v8.11.3
$ shower --help [COMMAND]
USAGE
  $ shower COMMAND
...
```

display help for shower

```

  $ shower help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
